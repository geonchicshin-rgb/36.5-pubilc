# 36.5 개인정보처리방침

시행일: 2026-05-29

36.5는 사용자의 작은 성공과 하루 기록을 돕는 회복 루틴 앱입니다. 본 방침은 36.5가 어떤 정보를 저장하고, 어떻게 보호하는지 설명합니다.

## 1. 수집 및 저장하는 정보

36.5는 사용자가 앱 안에서 직접 입력한 다음 정보를 기기 내 저장소에 저장할 수 있습니다.

- 온보딩 답변과 응원 온도 결과
- 목표, 세부 계획, 오늘의 한 줄
- 아침/저녁 기록, 정화 일기, 기분 태그, 복귀 행동
- 오늘의 30분 시작/완료/중단 기록
- 앱 설정, 알림 관련 선택값

## 2. 저장 위치

36.5의 데이터는 기본적으로 사용자의 기기 안에 저장됩니다. 현재 버전은 별도 서버 계정, 클라우드 동기화, 외부 데이터베이스를 사용하지 않습니다.

## 3. 제3자 제공

36.5는 사용자의 기록 데이터를 제3자에게 판매하거나 제공하지 않습니다. 사용자가 직접 백업/공유 기능을 사용하는 경우, 해당 데이터는 사용자가 선택한 공유 경로를 통해 이동할 수 있습니다.

## 4. 알림 권한

36.5는 사용자가 설정한 약속·30분 집중 시간을 떠올리기 위해 기기의 로컬 알림 권한을 요청할 수 있습니다. 알림은 보조 도구이며, 거부하셔도 메인 화면을 직접 열어 동일한 흐름을 사용할 수 있습니다. 알림 권한은 운영체제 설정 또는 앱의 ‘설정 → 약속 알림’에서 언제든 다시 변경할 수 있습니다. 모든 알림은 사용자 기기 안에서만 예약되며 외부 서버를 거치지 않습니다.

## 5. 데이터 삭제

사용자는 앱 설정의 전체 초기화 기능을 통해 기기 안에 저장된 데이터를 삭제할 수 있습니다. 앱 삭제 시 운영체제 정책에 따라 로컬 데이터도 함께 삭제될 수 있습니다.

## 6. 의료 및 상담 서비스 아님

36.5는 의료, 정신건강 진단, 치료, 상담 서비스를 제공하지 않습니다. 앱의 진단 결과와 코칭 문구는 자기이해와 일상 루틴 형성을 돕기 위한 참고 정보입니다. 심각한 불안, 우울, 자해 충동 등 긴급한 상황에서는 전문기관 또는 주변 사람에게 즉시 도움을 요청하세요.
